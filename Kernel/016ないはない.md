# 016ないはない  
  
　目標として最も駄目なものは何か。  
　高すぎる目標か、低すぎる目標か。  
  
　最も駄目な目標とは「◯◯しない」というものである。  
　たいてい、◯◯にはネガティブな、低いレベルの行動が当てはまるだろう。  
　人は[視線を向けた方向に進む](007視線を向けた方向に進む)ので、それは今のレベルあるいは更に低いレベルに目標を立てているに過ぎない。  
  
　失敗を恐れる者は、同じ失敗を何度も繰り返す。  
　それは失敗を見つめているからだ。  
　成功を見つめなければいけない。
