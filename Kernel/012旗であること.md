# 012旗であること

　様々なリーダー論がある。
　他者をまとめ、他者を正しく管理し、組織として成長を続けるためのノウハウが様々な形で広まっている。
　しかし私からすると、それはリーダーではない。

　リーダーとは旗であって、管理者やマネージャーではないのだ。

　リーダーとは常に[[004こだわりこそが、自分を形作る|強固なパーソナリティーを持ち]]、[[007視線を向けた方向に進む|高い目標に向かって]]、[[001好きこそが私の全て|好きなことをやり続ける]]人を言う。
　[[008他者は変えられない|その姿に共感した人が自然と集まり]]、自然と組織になっていく。その過程で管理者やマネージャーという役割が発生するのであって、リーダーは人々の憧れであることが仕事なのだ。
