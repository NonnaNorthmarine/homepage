# 012旗であること  
  
　様々なリーダー論がある。  
　他者をまとめ、他者を正しく管理し、組織として成長を続けるためのノウハウが様々な形で広まっている。  
　しかし私からすると、それはリーダーではない。  
  
　リーダーとは旗であって、管理者やマネージャーではないのだ。  
  
　リーダーとは常に[強固なパーソナリティーを持ち](004こだわりこそが、自分を形作る)、[高い目標に向かって](007視線を向けた方向に進む)、[好きなことをやり続ける](001好きこそが私の全て)人を言う。  
　[その姿に共感した人が自然と集まり](008他者は変えられない)、自然と組織になっていく。その過程で管理者やマネージャーという役割が発生するのであって、リーダーは人々の憧れであることが仕事なのだ。
